# AI 工具部署支持笔记

岗位里提到 AI 工具资源部署与应用支持。对实习生来说，先理解三个重点。

## 1. 数据安全

公司内部文档、设计数据、测试数据通常不能直接发到外部服务。

所以 AI 工具要考虑：

- 数据来源
- 权限控制
- 日志脱敏
- 访问审计
- 是否可以联网

## 2. AI 工具不是只有聊天

更常见的落地点是：

- 文档检索
- 日志摘要
- 报告生成
- 脚本模板生成
- QA 检查建议

本项目的 `flowctl ask` 是一个极简关键词检索助手。

```bash
flowctl ask "怎么看日志"
```

它没有调用大模型，但模拟了“先从内部知识库找答案”的思路。

## 3. 部署时要关心环境

真实部署要确认：

- 运行在哪台机器
- 用哪个 Python 环境
- 是否需要 GPU
- 能否访问内部文档
- 用户权限怎么控制
- 生成内容是否要留痕
